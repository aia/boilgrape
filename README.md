## BoilGrape ##

BoilGrape is a JRuby/Grape boiler plate.

### Technologies Used ###

* [JRuby](http://jruby.org/)
* [Puma](http://puma.io/)
* [Grape](http://intridea.github.io/grape/)
* [Rack](http://rack.github.io/)
* [RSpec](http://rspec.info/)

### Project Layout ###

├── app  
│   └── REST API code  
├── bin  
│   └── Project binaries  
├── config  
│   ├── Project configuration  
├── features  
│   ├── Cucumber BDD  
├── lib  
│   └── Project library  
├── rackup  
│   └── Rack configuration  
└── spec  
│   └──  Project tests
